# Homebrew Formula for Prism Scanner

## Installation

```bash
brew tap prismlab/tools https://github.com/prismlab/prism-scanner
brew install prism-scanner
```

## Alternative: Direct Formula

```bash
brew install --formula Formula/prism-scanner.rb
```

## Note

This formula is also published in the [prismlab/homebrew-tools](https://github.com/prismlab/homebrew-tools) tap.
