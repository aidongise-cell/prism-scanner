# prism-scanner

> Security scanner for AI Agent skills, plugins, and MCP servers.

This is an npm wrapper for [Prism Scanner](https://github.com/prismlab/prism-scanner). It requires Python 3.10+ to be installed on your system.

## Usage

```bash
# One-time scan (no install needed)
npx prism-scanner scan https://github.com/user/skill-repo

# Or install globally
npm install -g prism-scanner
prism scan ./my-skill/
```

## Requirements

- Node.js 16+
- Python 3.10+ (auto-detected)
